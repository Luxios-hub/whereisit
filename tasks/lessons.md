# Lessons

## 2026-07-08 — Design mockups must be rendered, not sketched
Correction: user rejected flat CSS "vibe" mockups as lifeless sketches.
Rule: never present flat colored-box mockups. Build the real rendered component
(SVG filters, gradients, lighting, texture, procedural sprites) and present
screenshots of the actual running UI. Target: "looks like a real dev team built
it", unique identity — not the default minimal-card aesthetic.
